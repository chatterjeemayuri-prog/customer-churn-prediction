# customer-churn-prediction

**Project Overview**
This project demonstrates an end-to-end machine learning workflow for predicting customer churn. The objective is to identify customers at risk of leaving and provide actionable insights to support retention strategies.
The project mirrors real-world business problems where predictive modelling must balance statistical rigor, interpretability, and practical implementation.

**Objectives**
- Perform exploratory data analysis (EDA)
- Engineer relevant customer behaviour features
- Build and compare classification models
- Evaluate model performance using appropriate metrics
- Interpret model outputs for business decision-making

**Tools & Methods**
- Python (Pandas, NumPy, Scikit-learn)
- Exploratory Data Analysis
- Logistic Regression
- Tree-based Models
- Model Validation (Cross-validation, ROC-AUC)
- Feature Importance Analysis

**Structure**
- data/ -- dataset files
- notebooks/ -- exploratory analysis and modelling notebooks
- src/ -- reusable scripts and functions
- README.md -- project documentation

**Business Impact**
Accurate churn prediction enables proactive retention strategies, optimized marketing interventions, and improved customer lifetime value.
